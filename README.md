# Tarides Company Handbook

This repository contains the **Tarides Company Handbook** - a publicly visible resource that documents our company's policies, practices, and culture.

## About This Handbook

This handbook is publicly visible - anyone can read and learn from our practices.

## Contributing

We welcome contributions from anyone! Please see
[CONTRIBUTING.md](./CONTRIBUTING.md) for details on how to contribute.

## Building and Linting

This repository includes a build script that lints all Markdown files to
ensure consistent formatting with 80-character line wrapping:

```bash
# Install dependencies (first time only)
npm install

# Run the linter
./build
# or
npm run build
```
